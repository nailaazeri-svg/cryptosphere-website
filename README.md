# Cryptosphere Website

Cryptosphere is a modern cryptocurrency landing page built using **HTML, CSS, and Bootstrap**.  
The project showcases a clean interface for exploring cryptocurrency information, market trends, and latest crypto news.

## Features

- Responsive design
- Cryptocurrency showcase section
- Market trend cards
- Latest crypto news section
- Contact form
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome
